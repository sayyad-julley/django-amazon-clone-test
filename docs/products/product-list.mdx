## Product List Table

The product list table provides an overview of all products with the following features:
- Displays product details including ID, Name, Subcategory, URL Slug, Status, and Creation Date
- "Actions" column with two buttons:
  1. "View" button: Navigate directly to product edit page
  2. "Add Media" button: Add media to the specific product

### View Details Link
- Located in the "Actions" column
- Links to the product edit page using the product's unique ID
- Allows quick access to edit product details
