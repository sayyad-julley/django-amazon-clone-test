# List Page Total Count Feature

## Overview
This feature adds a total count display to list page headers for categories, subcategories, products, and merchants.

## Implementation Details
- Added total count display using `{{ list_variable|length }}` in page titles
- Locations updated:
  - `category_list.html`
  - `sub_category_list.html`
  - `product_list.html`
  - `merchant_list.html`

## Usage Example
```html
<!-- Example from category_list.html -->
<h4>Category List <small>(Total: {{ categories_list|length }})</small></h4>
```
