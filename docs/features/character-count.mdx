## Character Count Functionality

The product create and edit forms include an automatic character count display for description text areas.

### Features
- Real-time character count updates
- Maximum limit of 500 characters
- Display format: "Current Count/Maximum Limit"

### Implementation Details
- Uses vanilla JavaScript event listeners
- Automatically updates as user types
- Provides visual feedback on character usage
