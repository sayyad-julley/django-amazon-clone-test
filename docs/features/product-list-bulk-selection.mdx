## Bulk Product Selection Feature

### Overview
The product list page now supports bulk selection of products using a "Select All" checkbox.

### Functionality
- A "Select All" checkbox is added to the product list page
- Clicking the checkbox selects or deselects all product checkboxes
- Individual product checkboxes can still be selected independently
- The checkbox state dynamically updates based on individual checkbox selections

### Usage
1. Navigate to the product list page
2. Use the "Select All" checkbox in the header to quickly select or deselect all products
3. Alternatively, select individual products as needed

### Implementation Details
- Located in: `DjangoEcommerceApp/templates/admin_templates/product_list.html`
- JavaScript handles checkbox state management
