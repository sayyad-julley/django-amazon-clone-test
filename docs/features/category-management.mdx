## Category and Subcategory List Pages

### Filter Functionality

The category and subcategory list pages now include a "Clear" button next to the search input field. This button provides a quick way to reset the current filter and show all items.

#### Features
- One-click filter reset
- Preserves original list view
- Available in both category and subcategory list pages

#### Usage
Simply click the "Clear" button to remove any applied filters and return to the full list view.
