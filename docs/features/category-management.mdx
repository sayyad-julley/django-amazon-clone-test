## Clear Filter Functionality

In both the Category and Subcategory list pages, a "Clear" button is provided next to the search input. This button allows users to quickly reset the filter and view all items.

### Features
- Instantly clears any applied filters
- Resets the list to show all items
- Located in both category and subcategory list views

### Usage Example
1. Apply a filter in the search input
2. Click the "Clear" button
3. The list will reset to show all items
