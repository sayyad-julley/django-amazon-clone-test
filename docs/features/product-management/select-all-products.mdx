## Select All Products Checkbox

### Overview
The product list page now includes a "Select All" checkbox functionality that allows users to easily select or deselect all products in the list.

### Features
- A "Select All" checkbox in the product list header
- Individual checkboxes for each product
- Dynamic selection behavior:
  - Clicking "Select All" checks/unchecks all individual product checkboxes
  - If all individual checkboxes are manually checked, the "Select All" checkbox automatically gets checked

### Usage
1. Navigate to the product list page
2. Use the "Select All" checkbox in the header to quickly select or deselect all products
3. Alternatively, select/deselect products individually

### Implementation Details
- JavaScript event listeners handle the checkbox interactions
- Provides a seamless user experience for bulk product selection

### Example
```html
<input type="checkbox" id="selectAllProducts"> Select All Products
```

### Best Practices
- Useful for operations like bulk edit, delete, or export of products
- Enhances user productivity by reducing manual selection efforts
