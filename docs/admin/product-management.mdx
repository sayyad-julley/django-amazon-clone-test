# Product Management

## Overview

The admin panel provides comprehensive product management capabilities including creating, editing, and managing products in the e-commerce system.

## Product Create Page

The product create page (`/admin/product/create`) allows administrators to add new products to the catalog.

### Features

- **Product Basic Details**: Name, brand, URL slug, category selection
- **Pricing**: Max price and discount price fields
- **Descriptions**: Short and long product descriptions with rich text editor
- **Product Media**: Support for multiple images and videos
- **Product Details**: Customizable title/details pairs
- **Product About**: About section with multiple entries
- **Product Tags**: Tag input for categorization
- **Navigation**: "Back to Product List" button for easy navigation

### Navigation

A "Back to Product List" button is available at the bottom of the form, allowing administrators to quickly return to the product listing without saving changes. The button uses Bootstrap styling and includes a left arrow for visual indication.

```html
<a href="{% url 'product_list' %}" class="btn btn-secondary btn-block">
  ← Back to Product List
</a>
```

## Product Edit Page

The product edit page (`/admin/product/edit/<product_id>`) allows administrators to modify existing products.

### Features

- All features from the create page, pre-populated with existing product data
- Product ID displayed in the header
- Existing product details, about sections, and tags are loaded
- **Navigation**: "Back to Product List" button for easy navigation

### Navigation

Similar to the create page, a "Back to Product List" button is available at the bottom of the form for quick navigation back to the product listing.

## Usage Examples

### Creating a New Product

1. Navigate to **Products** > **Add Product** from the admin sidebar
2. Fill in the required product information
3. Add media content (images/videos)
4. Add product details and about sections
5. Add relevant tags
6. Click "ADD PRODUCTS" to save
7. Use "← Back to Product List" to return to the listing without saving

### Editing an Existing Product

1. Navigate to **Products** > **Product List** from the admin sidebar
2. Click on a product to edit
3. Modify the desired fields
4. Click "EDIT PRODUCTS" to save changes
5. Use "← Back to Product List" to return to the listing
